# Todo app

App to follow up on your Todos.

Add, edit, delete or mark as done the Todos.

## Installation

1. Clone this repo

   ```bash
     git clone https://github.com/rusomarques/todo-app.git
     cd todo-app
   ```

2. Install dependencies & and start adding todos:

   ```bash
   npm install
   npm start
   ```

### Tech Stack

- React / Redux
